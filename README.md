# My Anime Watchlist

A web application that will allow a user to create an account and keep track of any anime they are currrently watching or have completed watching.

### Technologies Used
- HTML5
- CSS3
- JavaScript
- EJS
- Express
- Mongoose
- Node
- jQuery
- Google Fonts
- MongoDB
